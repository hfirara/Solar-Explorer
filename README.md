# Solar Explorer
Solar Explorer is an educational game developed using Unity (C#) where players explore the solar system through a structured, level-based gameplay experience. The game focuses on learning planetary information by exploration, information collection, and quiz-based challenges.

The game blends educational content with simple gameplay mechanics to encourage active learning and player engagement.

## Game Overview
- Genre: Educational / Exploration
- Platform: PC
- Engine: Unity
- Programming Language: C#
- Target Audience: Junior High School students and general players interested in astronomy

## Features
- Level-based exploration across 8 planets
- Planet information collection system
- Information storage system to track collected planetary facts
- Quiz challenges at the end of each planet level
- Final level with vertical movement mechanics combined with question-based challenges
- User interface system (menus, panels, volume control)
- Health/damage system triggered by incorrect answers
- Quest system to guide player progression
